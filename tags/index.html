      Where to put the imported save back

      Import to save hands you the merged file as a download — browsers can't overwrite the save in place. When your browser asks where to save, pick the folder below and save it over the old file (back it up first). If your browser adds a (1) to the name, delete it so it replaces the original.

      On Windows the save folder is here (under C:\Users\<you>\AppData\Local):

      %LOCALAPPDATA%\Rivals2\Saved\SaveGames

      Tip: paste this into Explorer's address bar, then replace Rivals2_PlayerTagSaveSlot.sav with the file you just downloaded.
