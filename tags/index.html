        or upload tag files

        Already have .r2tag files (from "Just download the tag files", or shared directly)? Add them here — they merge into the save in step 2 along with anything selected above.

        • 2

          Merge them into the setup's save file

          Load the save file from the setup you're updating. The selected tags are merged into it right here in your browser, and the updated save comes back as a download.

        • 3

          Put the updated save back

          Browsers can't overwrite the save in place, so the merged file arrived as a download. Move it into the save folder below, replacing the old file.

          On Windows the save folder is here (under C:\Users\<you>\AppData\Local):

          %LOCALAPPDATA%\Rivals2\Saved\SaveGames

          Tip: paste this into Explorer's address bar, then replace Rivals2_PlayerTagSaveSlot.sav with the file you just downloaded.

        Running setups somewhere without internet?

        Grab the offline installer: a single HTML file that merges .r2tag files into a save entirely on the machine — no connection needed. Download your entrants' tag files above while you're online, take both to the venue, and open the installer in any browser.
