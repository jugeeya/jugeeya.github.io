Submit a tag

Load your Rivals II save file, pick the tags you want to share, and link each to a start.gg account. Your save is read in your browser and never uploaded; submitting adds to the tag database for download by your TOs in the section below.

On Windows your save is here (under C:\Users\<you>\AppData\Local):

%LOCALAPPDATA%\Rivals2\Saved\SaveGames\Rivals2_PlayerTagSaveSlot.sav

Tip: paste this into the file picker's File name box and hit Open to jump straight to it.

    Each tag needs a start.gg account linked (right on its row above). Search a gamer tag or paste a profile URL; this is what lets a tag be pulled by bracket.
